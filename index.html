About Net Spectre

Net Spectre allows network professionals and enthusiasts to define and execute network capture and scan configurations with a simple, readable syntax. It empowers users to automate and streamline networking tasks.

Example of Net Spectre Code

config_capture : cap1
    interface : Ethernet
    packets : 60
    ipv4 : 100.110.26.162
    json : False
    ek : False


config_scan : scan2
    target : 100.110.26.162
    ports : 0-443
    arguments :
    verbose_level : 5


capture : cap1
scan : scan2
